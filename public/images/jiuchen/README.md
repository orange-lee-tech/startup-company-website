# jiuchen 品牌与固定资产图片说明

本目录存放九辰官网的品牌级图片与长期留存图片。

## 当前文件

- `background.png`：官网首页与关键模块使用的背景图。
- `jiuchen-logo-notext.png`：九辰 Logo 图形资产。
- `wj.qq.com.png`：腾讯问卷二维码，尺寸约 180×180；当前用于联系页“扫码咨询 / 关注”区域。
- `jiuchenedu.com.png`：域名相关留档图片，作为网站固定资产保存；当前不在页面展示。

## 维护原则

1. 公开页面引用路径应使用 `/images/jiuchen/文件名`。
2. 二维码更新时，应同步确认实际跳转目标是否仍为当前咨询问卷。
3. 固定资产留档图片可以保留在仓库中，但不要在页面中默认展示。
4. 新增品牌图、背景图、Logo 变体时，应在本文件补充用途和展示位置。
