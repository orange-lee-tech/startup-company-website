# images 图片资产目录说明

本目录用于存放官网图片类资产。图片会通过 `/images/...` 路径被网站访问。

## 目录结构

- `jiuchen/`：品牌级资产、背景图、Logo、咨询二维码、域名相关留档图片等固定资产。
- `contact/`：微信公众号、视频号、抖音等联系入口二维码。
- `case/`：学员案例相关图片或证明材料，必须完成授权与脱敏。
- `teacher/`：师资头像、卡片图、详情页图片，必须确认授权。

## 维护原则

1. 文件名尽量使用英文、小写、连字符或明确域名形式，避免空格和中文特殊符号。
2. 公开展示的图片必须确认版权、肖像权、授权和脱敏状态。
3. 固定资产类图片可以留存但不一定展示。
4. 删除或移动图片前，应先全文搜索引用路径，避免线上图片 404。
5. 对二维码类图片，应记录用途、尺寸、指向链接和展示位置。
