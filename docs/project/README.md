# project 项目状态文档说明

本目录用于记录九辰教育官网的建设状态、内容口径、阶段性成果、待办事项和后续维护计划。

## 当前项目状态

官网已进入正式上线后的修缮与优化阶段。

已完成的基础事项包括：

- 域名上线：`https://jiuchenedu.com/`
- HTTPS 可用
- 非 www 主域规范化
- ICP 备案号展示：`湘ICP备2026021463号`
- 搜索引擎基础验证文件上线
- `robots.txt`、`sitemap.xml`、`llms.txt` 上线
- 联系页接入腾讯问卷
- 公开页面合规降敏
- 案例页按反馈弱化数量、强化初始情况与最终结果
- 基础缓存与安全响应头配置

## 内容口径

官网内容应保持：

- 专业、理性、可信赖
- 强调规划、评估、陪跑和复盘
- 避免夸张承诺和结果保证
- 案例采用脱敏展示
- 视觉上避免暗沉和信息堆叠

## 当前未完成事项

- 公安备案号下发后补入页脚。
- JSON-LD 结构化数据尚未加入。
- 搜索引擎后台需要在 sitemap 稳定后复检。
- 师资授权和案例脱敏台账需要持续维护。
- 旧模板页源码可择机清理。
- 健康巡检脚本可后续补充。

## 文档维护原则

1. 阶段性成果应记录“完成了什么”和“还剩什么”。
2. 内容反馈应转化为明确页面口径，例如“弱化案例数量，突出结果”。
3. 涉及授权、脱敏、备案的事项应单独标注状态。
4. 不要把临时聊天结论散落在对话中，应沉淀为可复用文档。
